## 简体中文版
当前版本：v1.0.0(Build 1.0.9695.13046)。

这是一个简单的控制台走迷宫程序。  
您需要通过 `W`、`A`、`S`、`D` 四个键控制小球 `O` 的移动，让小球 `O` 走到出口 `E` 处来完成关卡。  
尝试突破每一关的纪录！

### 使用方法
请您按照菜单上的提示操作。  
如果您需要切换语言，请输入 `3`，然后根据菜单提示操作。

## 繁體中文版
當前版本：v1.0.0(Build 1.0.9695.13046)。

這是一個簡單的主控台走迷宮程式。  
您需要透過 `W`、`A`、`S`、`D` 四個鍵控制小球 `O` 的移動，讓小球 `O` 走到出口 `E` 處來完成關卡。  
嘗試突破每一關的紀錄！

### 使用方法
請依照選單上的提示操作。  
如果您需要切換語言，請輸入 `3`，然後根據選單提示操作。
