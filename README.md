## 简体中文版
当前版本：v0.0.2-PreRelease(Build 0.0.9686.5818)。
> [!CAUTION]
> 预发布版本并不稳定，并且可能缺失功能，请谨慎安装。

这是一个简单的控制台走迷宫程序。

您需要通过 `W`、`A`、`S`、`D` 四个键控制小球 `O` 的移动，让小球 `O` 走到出口 `E` 处来完成关卡。

尝试突破每一关的纪录！

### 使用方法
请您按照菜单上的提示操作。

如果您需要切换语言，请输入 `3`，然后根据菜单提示操作。
